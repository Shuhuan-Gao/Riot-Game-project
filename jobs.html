Game Developer (Reference: G001)

Position Title: Senior Game Developer

Reports To: Lead Game Developer

Salary Range: $90,000 - $120,000 per annum

Position Description

We are seeking a talented Senior Game Developer to join our team. You will be responsible for developing and implementing game features, optimizing performance, and collaborating with designers and artists to create immersive gaming experiences.

Key Responsibilities

  1. Develop and implement game features using C++ and Unreal Engine
  2. Optimize game performance and memory usage
  3. Collaborate with designers and artists to implement game mechanics
  4. Write clean, maintainable, and well-documented code
  5. Participate in code reviews and provide technical guidance
  6. Debug and fix issues in existing game systems

Requirements

Essential
  • Bachelor's degree in Computer Science or related field
  • 5+ years of experience in game development
  • Strong proficiency in C++ and Unreal Engine
  • Experience with 3D mathematics and physics
  • Excellent problem-solving skills
Preferable
  • Experience with multiplayer game development
  • Knowledge of shader programming
  • Experience with version control systems (Git)
  • Portfolio of completed game projects

UI/UX Designer (Reference: U002)

Position Title: Senior UI/UX Designer

Reports To: Creative Director

Salary Range: $85,000 - $110,000 per annum

Position Description

We are looking for a creative UI/UX Designer to join our team. You will be responsible for creating intuitive and engaging user interfaces for our games, ensuring a seamless user experience across all platforms.

Key Responsibilities

  1. Design and prototype user interfaces for games
  2. Create wireframes, mockups, and interactive prototypes
  3. Conduct user research and usability testing
  4. Collaborate with developers to implement designs
  5. Ensure consistency across all game interfaces
  6. Stay updated with UI/UX trends and best practices

Requirements

Essential
  • Bachelor's degree in Design or related field
  • 4+ years of experience in UI/UX design
  • Proficiency in Figma, Adobe XD, or similar tools
  • Strong portfolio demonstrating UI/UX work
  • Understanding of game design principles
Preferable
  • Experience with motion design
  • Knowledge of HTML/CSS
  • Experience in game industry
  • Understanding of accessibility standards

QA Tester (Reference: Q003)

Position Title: Senior QA Tester

Reports To: QA Manager

Salary Range: $75,000 - $95,000 per annum

Position Description

We are seeking an experienced QA Tester to join our quality assurance team. You will be responsible for ensuring the quality of our games through systematic testing, bug reporting, and quality control processes.

Key Responsibilities

  1. Develop and execute test plans and test cases
  2. Identify and document software defects and issues
  3. Perform regression testing and verify bug fixes
  4. Collaborate with development teams to ensure quality standards
  5. Participate in agile development processes
  6. Create and maintain test documentation

Requirements

Essential
  • Bachelor's degree in Computer Science or related field
  • 3+ years of experience in QA testing
  • Strong understanding of software testing methodologies
  • Experience with bug tracking systems (JIRA, Bugzilla)
  • Excellent analytical and problem-solving skills
Preferable
  • Experience with automated testing tools
  • Knowledge of game development processes
  • Experience with performance testing
  • Understanding of agile development methodologies
